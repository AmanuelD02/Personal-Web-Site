Does it matter to use relative size?

Yes. Because absolute units such as cm, mm, in, pt and pc doesn't depend on the screen of the user it is always the same. And this way of designing website makes the website only usable for a specific screen size, the others like moblie will not be considered. Relative units have nothing to do with the element itself rather it is dependant with the screen.Therefore, the element can be used for different screen with different size and orientation. w3.org
