Specific changes in CSS3

There are quite a few new properties in CSS3.Below are some examples:

Color:
New way of describing color is introduced into CSS3. Color module, hexadecimal and rgb() was the only way to describe in previous version. In CSS3, the color keyword list has been extended to include 147 additional keyword colors , and also provides us with a number of other options such as HSL, HSLA, and RGBA. These includes the ability to declare semitransparent colors.

New selectors:
The evolution of selector:- In css1, id, class and element type was the only way. In CSS2, pseudo-elements, pseudo-classes, and combinators were introduced. Now, many selector types are added.
Example: nth-child, Element[attr^=val] ,Element[attr$=val] ...

Element[attr^=val]
Eg. a[href^=’https://piazza.com/.’]{color: green’} Matches any element E whose attribute a starts with the value https://piazza.com/

Element[attr*=val] Eg. img[src*=’gallery’]{border-color: #a7e7ec;} This match element with img attribute having image source that contains 'gallery' in any of the part.

New layouts:
Many new layouts are introduced in CSS3 that makes developing website easier and making it more apealing.

Examples:
display: table;
display: table-cell; This gives you a table layout without the HTML tables.
display: flexbox;
flex-order: 1;   With flexbox you have greater control over how the browser flows boxes, like divs onto the page.
